
## KpiCard Semantic Design Rules
When building or modifying KpiCard components in this project, strictly follow this semantic mapping for icons and colors:
- Target/Goal: Icon Target, Color var(--text)
- Forecast: Icon BarChart3 (total) / Activity (breakdown), Color var(--blue) (total), green(100%), teal(80%), amber(50%)
- Pipeline/Open: Icon FolderKanban, Color var(--violet)
- Won/Success: Icon Trophy or CheckCircle2, Color var(--green)
- Timeliness: Icon CalendarDays / Filter / PhoneCall, Color var(--blue) / var(--teal) / var(--amber)
- Conversion: Icon TrendingUp, Color var(--green)
- Inbox/Total: Icon Inbox / ListTodo, Color var(--blue) / var(--text)
- Error/Risk: Icon AlertTriangle / XCircle, Color var(--red) / var(--amber)
