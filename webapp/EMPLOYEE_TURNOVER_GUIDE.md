# คู่มือพนักงานเข้า-ออก กับ Target / Forecast / Actual

อัปเดต 2026-07-15 · คู่กับฟีเจอร์ "โอนงาน" ในหน้า ตั้งค่า → ผู้ใช้งาน (ปุ่ม ⇄)

**หลักจำ:** TG (เป้า) ผูกกับ "คน+เดือน" ในหน้าวางเป้า · FC (คาดการณ์) ติดไปกับ*ดีลที่เปิดอยู่* ·
AT (ยอดจริง) ติดกับ*ดีลที่ปิด Won แล้ว* → FC โยกด้วยการโอนดีล, TG โยกด้วยการแก้เป้า, **AT ไม่โยกเลย**

**นโยบาย:** เดือนที่ผ่านมาแล้วไม่โยกอะไรทั้งสิ้น — ประวัติคงใต้ชื่อเดิมจนจบปี
ปีใหม่รีเซ็ตด้วยรายชื่อคนที่ active เท่านั้น (ระบบจัดการให้เอง)

---

## 🟢 พนักงานเข้า

1. **สร้างบัญชี** — ตั้งค่า → ผู้ใช้งาน → เพิ่มผู้ใช้ (กำหนดบทบาท + ทีม)
2. **ใส่เป้า** — หน้าวางเป้า → กรอกเป้ารายเดือน**ตั้งแต่เดือนที่เริ่มงาน**
   (เดือนก่อนหน้าปล่อยว่าง — % รายปีจะได้ไม่โดนหารด้วยเดือนที่ยังไม่ได้ทำงาน)
3. **FC ไม่ต้องทำอะไร** — เกิดเองเมื่อสร้าง/รับโอนดีล
4. **รับพอร์ตต่อจากคนเก่า** — ใช้ปุ่ม "โอนงาน" (ดูฝั่งออก)

## 🔴 พนักงานออก

1. **ระงับบัญชีทันที** — ตั้งค่า → ผู้ใช้งาน → ปุ่มแม่กุญแจ (ปิดบัญชี **ห้ามลบ**)
   → เข้าระบบไม่ได้ทันที, หายจาก dropdown ทุกจุด, ข้อมูล/ประวัติอยู่ครบ
2. **กดปุ่ม "โอนงาน" (⇄)** ที่แถวของคนออก → เลือกผู้รับ → ระบบทำให้ครบในคลิกเดียว:
   - **ดีลที่ยังเปิดทั้งหมด** เปลี่ยนผู้ดูแล+ทีมเป็นคนรับ → FC เดือนปัจจุบัน/อนาคตย้ายตามทันที
   - **เป้าตั้งแต่เดือนที่เลือก** (ค่าเริ่มต้น = เดือนถัดไป) โยกไปบวกให้คนรับ ต้นทางเหลือ 0
   - **ดีลที่ Won/Lost แล้ว + เป้าเดือนเก่า: ไม่ถูกแตะ** — บันทึก audit ครบว่าโอนอะไรบ้าง
3. เสร็จแล้วตรวจหน้าวางเป้า 1 รอบ — ถ้าอยากเกลี่ยเป้าที่รับมาให้หลายคน แก้มือต่อได้เลย

### เดือนคาบเกี่ยว (ออกกลางเดือน)
AT ที่ปิดไปแล้วอยู่กับคนเก่า / FC ดีลเปิดย้ายไปคนใหม่ → % ต่อเป้าเดือนนั้นดูแบ่งกันสองคน
**แนะนำ: ปล่อยไว้แล้ววัดเดือนนั้นที่ระดับทีมแทน** (ค่าเริ่มต้นของปุ่มโอนงานจึงโยกเป้าตั้งแต่
"เดือนถัดไป" — ถ้าอยากโยกเดือนปัจจุบันด้วย เปลี่ยนเดือนเริ่มในโมดัลได้)

## 🔄 ย้ายทีม

1. แก้ทีมที่บัญชีผู้ใช้ (หน้า ผู้ใช้งาน → แก้ไข)
2. ดีลที่ยังเปิด: แก้ทีมของดีลให้ตามไป (ใช้ปุ่มโอนงานโอนให้ "ตัวเอง" ไม่ได้ — แก้ทีละดีล
   หรือแจ้งผู้ดูแลระบบ) — ดีลเดือนเก่ารายงานใต้ทีมเดิม ถูกต้องแล้ว
3. เป้าเดือนอนาคต: ย้ายจากตารางทีมเก่าไปทีมใหม่ในหน้าวางเป้า

## สิ่งที่ระบบจัดการให้อัตโนมัติ (ไม่ต้องทำมือ)

- คนถูกระงับหายจาก dropdown มอบหมายงาน + wizard แจกเป้าปีใหม่ (PR #401)
- หน้าวางเป้า: เป้าค้างของคนออกโชว์เป็นแถว "ออกจากระบบแล้ว" (เกลี่ยออกได้) และหายเองเมื่อยอดเป็น 0/ขึ้นปีใหม่
- หน้า KPI ดีล: แถวรายบุคคลของคนออกติดป้าย "ออกจากระบบแล้ว" (PR #402) และหายเองปีถัดไป
- เอกสารโครงการเก่ายังเติมเบอร์/อีเมลของ AE ที่ออกไปแล้วได้
